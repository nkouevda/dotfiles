# dotfiles

My dotfiles.

See also [nkouevda/bin](https://github.com/nkouevda/bin).

## Installation

See [`Makefile`](Makefile) for available targets. For example:

    make bash bat ctags curl dig dircolors fzf git karabiner kitty less readline ripgrep ssh tig vim

To copy instead of symlinking, use `cp=1`:

    make cp=1 ...
